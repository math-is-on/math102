** Solving Trig Equations

Now we want to talk about how we might solve the equation
\[
\sin^2 (x) - 2\sin(x) + 1 = 0\]
We'll build up to this equation, but first let's start with some simple examples.
\vspace{10in}

** Example
Solve the equation
\[
\sin(x) = \frac{1}{2}\]
for \( 0 \le x < 2\pi\)

\vspace{10in}

** Example

** Example
Solve the equation
\[
2\cos(x) - \sqrt{2} = 0 \]
for \( 0 \le x < 2\pi\).
